# eeg-spectral-shift
This script computes the EEG power in canonical frequency bands using EEGLAB toolbox (MATLAB) and exports the results to an excel sheet. 

Part of the frequency-band extraction workflow is based on standard EEGLAB spectopo examples and EEGLAB mailing-list guidance.

- Bands - delta, theta, alpha, beta, delta, gamma (30-80Hz)
- Power units : absolute and dB
